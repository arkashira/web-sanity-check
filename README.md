# Web Sanity Check

This project provides a simple web sanity check tool that records user flows and generates tests automatically.

## Usage

1. Create a `UserFlow` object with the desired steps.
2. Record the user flow using the `record_user_flow` method.
3. Generate tests using the `generate_tests` method.
4. Save the recorded flows to a file using the `save_recorded_flows` method.
5. Load the recorded flows from a file using the `load_recorded_flows` method.

## Testing

Run the tests using `pytest`.
